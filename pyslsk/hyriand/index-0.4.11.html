SLSK-UL: My PySoulSeek Upload Bandwidth Limiting Patch

Latest release: version 3
Changes in this version: added banning of users
 
This patch adds additional options to the configuration screen (in the transfers tab) which allows you to control the maximum upload bandwidth used by PySoulSeek in KILO-BYTES (not bits) per second. The maximum upload speed per upload is the value you set divided by the number of uploads. The upload bandwidth limit can be altered during a transfer. It will also allow you to select a fixed number of upload slots instead of the automatic queueing which is used in the original PySoulSeek. It also implements a banning function.
 

SLSK-UI: My PySoulSeek User Interface Enhancement Patch

Latest release: version 4.3
Changes in this version: Human readable sizes fixed, undocumented num-lock-turns-off-auto-completion feature removed
 
This patch adds some additional features to the user interface of PySoulSeek. It is intended to increase the overall niftyness of the user interface :-)
 
Feature quicklist:
  • (Partial) automatic nickname completion in the chat input field (press the Tab key)
  • Meta-commands in the (private) chat input field (see below)
  • Human readable sizes in the transfer windows (sizes shown in 2 decimal figures in Gb's, Mb's or Kb's)
  • An "Add to List" and a "Browse" button on the user info pages
  • Total upload/download speeds in the status bar
 
Meta Commands
The slsk-ui patch implements a couple of new chat meta-commands. It's a bit like IRC but with specific SoulSeek commands. Some of them only work in chatrooms, others only in private chats and most of them work in both places (see the notes).  
command arguments explanation notes
/add /ad nickname Add the specified user to your user list 1
/away /a   Toggles your away status 1
/browse /b username Browse files of the specified user 1
/close /c   Closes the active private chat window 3
/join /j channel Join the specified channel 2
/leave /l [channel] Leave the current or specified channel (channel argument is optional) 2
/msg /m username message Open a private chat window to tje specified user and send a message. Username argument can optionally be single or double quoted (for nicknames with spaces in them) 2
/part /p [channel] See /leave 2
/pm username Open a private chat window to the specified user (but don't send a message, no need to quote the username if there are spaces in it) 2
/quit /q   Quit PySoulSeek 1
/whois /w username View the user info of the specified user 2

Notes:
1) Works in both chatrooms and private chats (in private chats username arguments are optional)
2) Only works in chatrooms
3) Only works in private chats
